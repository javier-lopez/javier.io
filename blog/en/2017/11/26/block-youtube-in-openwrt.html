block youtube by IP in openwrt routers

26 Nov 2017

In previous posts I wrote about how to install openwrt as an access point or as a wireless repeater, this time Ill show how to block youtube and other third party sites by ip. The procedure works in desktop / and mobile devices.

The target router is a TP-Link N750, and Im using the latest trunk build.

OpenWRT uses UCI to centralize configuration, firewall rules are located at:

  • /etc/config/firewall

In order to block sites by IP youll need to modify such file appending the desired rules, eg. for blocking YouTube:

config rule
	option name		Block-YouTube-187.189.89.77/16
	option src		lan
	option family		ipv4
	option proto		all
	option dest		wan
	option dest_ip		187.189.89.77/16
	option target		REJECT
