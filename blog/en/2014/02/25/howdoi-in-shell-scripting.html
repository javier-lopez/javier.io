howdoi, a code search tool and a sh implementation

25 Feb 2014

During these days I read about howdoi, a stackoverflow client for your terminal. And I though it was pretty cool, so I looked at a couple of implementation (the original in python and a clone in ruby) and decided to do my own version, I’ve just learned awk and wanted something where I could use it. Besides, this version doesn’t require anything but awk and wget.

Get the code at:

  • https://raw.github.com/javier-lopez/learn/master/sh/tools/howdoi
